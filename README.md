## 03. 운영 철학

- **Public-safe boundary**: 공개 레포와 내부 운영을 분리해 외부 노출 경로를 최소화.
- **Composable design**: 공개 레이어는 독립 모듈처럼 연결.
- **Evidence-ready structure**: 산출물과 규약을 한눈에 연결 가능한 구조로 정리.
